# iPad 提詞器工作室

專為 iPad 和手機設計的網頁版提詞器應用程式，具備相機錄影功能。

## 功能特色

- 📝 **講稿顯示** - 自動滾動提詞器文字
- 📹 **相機錄影** - 使用內置相機錄製影片
- ⏱️ **錄影計時器** - 錄影時顯示計時器
- 🔄 **鏡像模式** - 翻轉文字（適用於分光鏡設置）
- 🌐 **多語言** - 英文 / 繁體中文
- 📱 **響應式設計** - 針對 iPad 和 iPhone 優化

## 使用方法

1. **打開** `index.html` 在 iPad 或 iPhone 的 Safari 瀏覽器
2. **點擊「開啟鏡頭」** 預覽相機畫面
3. **點擊「全螢幕」** 進入錄影模式
4. **編輯講稿** 在講稿文字區域
5. **點擊「開始錄製」** 開始錄影和自動滾動
6. **點擊紅色「停止錄製」按鈕** 完成錄影
7. **下載** 你的影片

## 控制按鈕

| 按鈕 | 描述 |
|------|------|
| 開啟鏡頭 | 開啟相機預覽 |
| 切換鏡頭 | 切換前/後鏡頭 |
| 開始錄製 | 開始錄影 |
| 停止錄製 | 停止並保存錄影 |
| 開始滾動 | 開始自動滾動講稿 |
| 重置 | 重置滾動位置到頂部 |
| 鏡像文字 | 水平翻轉文字 |
| 速度滑桿 | 調整滾動速度 |

## 語言切換

點擊左上角的 **中文 / EN** 按鈕切換英文和繁體中文。

## 系統要求

- 現代網頁瀏覽器（iOS 推薦使用 Safari）
- 需要相機和麥克風權限

## 使用提示

- 全螢幕運行以獲得最佳體驗
- 視線保持在鏡頭附近以獲得自然的眼神接觸
- 調整滾動速度以配合你的閱讀節奏
- 配合提詞器分光鏡使用鏡像模式
